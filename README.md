# Jenkins Configuration as Code Learning Project

This repository now contains a JCasC-based Jenkins setup for the expanded task list:
- GitHub OAuth auth
- Role-based authorization
- Team-scoped access
- Seed job DSL
- Pipeline generation
- Multibranch jobs
- Shared libraries
- Agents
- Credentials

## Run

```bash
docker compose up --build -d
```

Open Jenkins at http://localhost:8082.
